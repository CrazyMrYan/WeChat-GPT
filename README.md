## 技术栈
### 前端
- Next
- HTML
- CSS

### 后端
- Node
- Koa
- PM2
- Sequelize

### 数据库
- MySQL

- 未混淆源码（可完美二开）
- 后端接口文档
- 初始化 SQL 文件

## 需准备
- 4核8G 或以上配置服务器
- 商户号
- 备案域名
- 公众号（非订阅号）
